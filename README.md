# To-do-List-App

A robust, scalable, and user-friendly To-Do List application developed with **Flutter** and **Dart**. The entire UI/UX was prototyped and refined in **Figma** prior to implementation, ensuring a seamless user experience.

This repository demonstrates the application of modern software engineering practices and agile methodologies, making it both a functional tool and a comprehensive educational resource for system design.

## Key Features & Architecture

* **UI/UX Design:** Fully prototyped using Figma.
* **Framework:** Built with Flutter and Dart.
* **Architecture:** Implements a strict **Feature-First Architecture** to maintain high cohesion, low coupling, and scalable code.
* **Design Patterns:** Utilizes the **Service Locator** pattern for efficient dependency management.
* **Methodology:** Developed under Agile principles using Scrum, structured User Stories, and a prioritized Product Backlog.
